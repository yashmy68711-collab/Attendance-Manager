# GUI Attendance Manager (Python)

## About

A simple attendance manager built using Python and tkinter.

## Features

* Mark student attendance
* Present/Absent option
* Save attendance to file
* GUI interface

## Tech Used

* Python
* Tkinter

## How to Run

Run the Python file and mark attendance.

## Future Improvements

* Search attendance
* Attendance history
* Better UI design
* Save by date
